# 🚀 LeetCode Journey: Learning & Progress

Welcome to my LeetCode repository! This project is a dedicated space where I track my daily progress, practice coding challenges, and build a strong foundation in Data Structures and Algorithms (DSA). 

## 🎯 Current Status
* **Goal:** Master core algorithmic patterns and prepare for technical interviews.
* **Current Focus:** Moving from foundational concepts to intermediate problem-solving.
* **Languages Used:**  C++

---

## 📚 Topics Completed

Here are the concepts and problem types I have mastered so far:

### 🔢 Basics
* Core syntax, loops, and conditional statements.
* Basic arithmetic algorithms and time complexity fundamentals.

### 🗂️ Arrays
* Iteration, filtering, and in-place transformations.
* Subarrays, prefix sums, and fixed-size windows.

### 🔤 Strings
* Character manipulation, ASCII operations, and string parsing.
* Palindrome checks, anagrams, and substring matching.

### ⚡ Bit Manipulation
* Understanding bitwise operators (`AND`, `OR`, `XOR`, `NOT`, `Shifts`).
* Solving problems related to masks, binary representations, and subset logic.

### 🔍 Binary Search
* Classic target searching in sorted spaces.
* Identifying search boundaries and optimizing $O(N)$ linear scans to $O(\log N)$.
